= Martin Desrosiers Dot Files

Setup my dot files for things like zsh, Vim, Git, etc.. I got the Rakefile from Ryan Bates's dotfiles project.

== Installation

=== Clone the project and install it

  git clone git://github.com/mdesrosiers/dotfiles ~/.dotfiles
  cd ~/.dotfiles
  git submodules init
  git submodules update
  rake install

=== Install Janus

  cd ~/.vim
  rake
